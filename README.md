# Sistema de Cadastro em Arquivos com Ordenação

Este trabalho tem como objetivo implementar um sistema funcional de cadastro onde é possível buscar informações sob diversos critérios, ordenar, exibir, editar, salvar e excluir dados contidos em um arquivo CSV de forma organizada e eficiente utilizando a linguagem C++. O projeto tem a finalidade de aplicar na prática os conceitos estudados na disciplina, como manipulação de arquivos, registros, vetores, alocação dinâmica de memória e modularização por meio de funções;
 
O tema escolhido foi “cantores”, onde foram utilizados o nome do cantor, seu sexo (feminino ou masculino), nacionalidade, ano de nascimento e gênero musical;
